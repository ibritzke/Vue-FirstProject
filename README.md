# Projeto básico de conhecimento em Vue

# Projeto
Projeto desenvolvido utilizando 
- Vue
- Axios
- VueRouter
- CSS

## Executando a aplicação
```
yarn install
```
```
yarn serve
```
Interaja com a aplicação em `http://localhost:8080/`
